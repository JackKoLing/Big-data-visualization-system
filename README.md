  # Big-data-visualization-system
（基于用户行为的数据可视化系统）
 
##  开发环境
 
 - jdk:1.7
 
 - 框架：spring 4.2.5 + mybatis 3.2.8

##  前提条件：

- 原始数据的收集：将app的每个页面的每个动作抽象为一个动作，将动作作为监听事件，加上事件的gps、发生时间

- 数据提取：从一个用户的总结 - 一群用户的总结 -所有用户的总结，最后将结论性数据放在数据库中
